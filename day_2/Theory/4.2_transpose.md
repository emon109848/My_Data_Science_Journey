## 1Ô∏è‚É£ What is Transpose?

* **Transpose of a matrix** ‚Üí flip rows and columns
* For 2D array: shape `(m,n)` ‚Üí `(n,m)`
* Useful for converting **row vectors ‚Üî column vectors**

---

## 2Ô∏è‚É£ NumPy Syntax

```python
import numpy as np

A = np.array([[1,2,3],[4,5,6]])
print(A.T)
# [[1 4]
#  [2 5]
#  [3 6]]
```

* `.T` ‚Üí shorthand for transpose
* Works for **2D and higher-dimensional arrays** (swaps last two axes)

---

## 3Ô∏è‚É£ Swapping Axes

* For higher dimensions, use `np.transpose`:

```python
B = np.array([[[1,2],[3,4]],[[5,6],[7,8]]])
B_transposed = np.transpose(B, (0,2,1))  # swap last two axes
```

* `(axis0, axis2, axis1)` ‚Üí new axis order

---

## 4Ô∏è‚É£ Row vs Column Vectors

```python
v = np.array([1,2,3])        # 1D row vector
v_col = v.reshape(-1,1)      # column vector
v_col.T                      # transpose ‚Üí row vector again
```

* **1D arrays** ‚Üí transpose has no effect ‚Üí reshape needed
* **2D arrays** ‚Üí transpose flips rows & columns

---

## 5Ô∏è‚É£ Practical Use Cases

1. **Matrix Algebra / Linear Algebra**

   * Compute `A.T @ A` ‚Üí used in least squares regression
   * Change vector orientation for multiplication

2. **Data Science / ML**

   * Feature matrix: samples as rows ‚Üí transpose to align with operations

3. **Graphics / Engineering**

   * Convert row coordinates to column coordinates for transformations

4. **Broadcasting / Reshaping**

   * Easily match shapes of arrays for operations

---

## 6Ô∏è‚É£ Quick Tips

* `.T` is simplest for 2D arrays
* `np.transpose(arr, axes=...)` ‚Üí control axis order in nD
* Use `.reshape(-1,1)` for column vectors before transpose if needed

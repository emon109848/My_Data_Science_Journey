## 1ï¸âƒ£ What is a Matrix Inverse?

* For a square matrix `A`, the **inverse** `Aâ»Â¹` satisfies:

\[
A \cdot A^{-1} = I
\]

* Only **invertible matrices** (non-singular) have an inverse.
* **Singular matrix:** `det(A) = 0` â†’ no inverse

---

## 2ï¸âƒ£ NumPy Syntax

```python
import numpy as np

A = np.array([[1, 2], [3, 4]])

# Check if invertible
det = np.linalg.det(A)
if det != 0:
    A_inv = np.linalg.inv(A)
    print(A_inv)
```

* `np.linalg.inv(A)` â†’ computes matrix inverse
* Returns **float ndarray** even for integer matrices

* Inverse used in **state-space models** to compute control signals

## 4ï¸âƒ£ Numerical Stability Tips

* Large or nearly singular matrices â†’ inverse may be **unstable**
* Prefer `np.linalg.solve()` instead of computing inverse explicitly for solving systems
* Check `np.linalg.cond(A)` â†’ condition number (large â†’ unstable)

---

## ğŸ”¥ Quick Tips

* Only **square, non-singular matrices** have inverse
* `.T` does NOT compute inverse (transpose â‰  inverse)
* Always check `det(A) != 0` or use `np.linalg.cond(A)` for stability

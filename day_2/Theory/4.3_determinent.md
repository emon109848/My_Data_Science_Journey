## 1Ô∏è‚É£ What is Determinant?

* Defined **only for square matrices** `(n x n)`
* **Geometric meaning:**

  * Measures **scaling factor** of linear transformation
  * Area/volume of parallelogram (2D) or parallelepiped (3D)
  * If `det = 0` ‚Üí matrix is **singular** (non-invertible)

---

## 2Ô∏è‚É£ NumPy Syntax

```python
import numpy as np

A = np.array([[1,2],[3,4]])
det = np.linalg.det(A)
print(det)   # -2.0
```

* Use `np.linalg.det(matrix)`
* Returns **float**, even for integer matrices

---

## 3Ô∏è‚É£ Practical Use Cases

1. **Check if Matrix is Invertible**

```python
if np.linalg.det(A) != 0:
    A_inv = np.linalg.inv(A)
else:
    print("Matrix is singular, cannot invert")
```

2. **Linear Algebra / Systems of Equations**

* Solve `Ax = b` only if `det(A) ‚â† 0`
* Used in **engineering** to check system solvability

3. **Geometry Applications**

* Area of triangle (2D) with points P1, P2, P3:

\[
Area = 0.5 * |det([P2-P1, P3-P1])|
\]

* 3D ‚Üí volume of parallelepiped

4. **Computer Graphics**

* Check if transformation matrix **flips / scales / collapses** space

---

## 4Ô∏è‚É£ Quick Tips

* Determinant only valid for **square matrices**
* Small matrices ‚Üí use formula
* Large matrices ‚Üí use `np.linalg.det()` (optimized)
* Determinant ‚âà 0 ‚Üí matrix is almost singular ‚Üí beware of numerical instability

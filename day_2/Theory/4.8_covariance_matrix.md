## 1Ô∏è‚É£ Definitions

1. **Variance**: Measures spread of a single variable

\[
\text{Var}(X) = \frac{\sum (X_i - \bar{X})^2}{n-1}
\]

2. **Covariance**: Measures how two variables **vary together**

\[
\text{Cov}(X,Y) = \frac{\sum (X_i - \bar{X})(Y_i - \bar{Y})}{n-1}
\]

* Positive ‚Üí move in same direction
* Negative ‚Üí move in opposite direction
* Zero ‚Üí no linear relationship

3. **Covariance Matrix**: Square matrix of covariances between variables

\[
\text{Cov}(X_1, X_2, ..., X_n) =
\begin{bmatrix}
\text{Var}(X_1) & \text{Cov}(X_1,X_2) & ... \
\text{Cov}(X_2,X_1) & \text{Var}(X_2) & ... \
\vdots & \vdots & \ddots
\end{bmatrix}
\]

4. **Correlation**: Standardized covariance ‚Üí ranges [-1,1]

\[
\text{Corr}(X,Y) = \frac{\text{Cov}(X,Y)}{\sigma_X \sigma_Y}
\]

## 2Ô∏è‚É£ NumPy Implementation

```python
import numpy as np

# Sample dataset (rows = variables, columns = samples)
X = np.array([[2,4,6,8],
              [1,3,5,7]])

# Covariance matrix
cov_matrix = np.cov(X)          # by default: rows = variables
print(cov_matrix)
```

* `np.cov(X, rowvar=True)` ‚Üí rows = variables, columns = samples
* `np.corrcoef(X)` ‚Üí correlation matrix

## 4Ô∏è‚É£ Quick Tips

* Diagonal ‚Üí variances
* Off-diagonal ‚Üí covariances
* Symmetric matrix ‚Üí cov(X,Y) = cov(Y,X)
* Standardize to correlation for **scale-invariant comparison**

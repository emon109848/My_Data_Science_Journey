# ğŸ”¹ 3ï¸âƒ£ `with` Statement (Context Manager)

Best Practice âœ…

```python
with open("data.txt", "r") as f:
    content = f.read()
```

Why use `with`?

* Automatically closes file
* Cleaner
* Safer

No need for `f.close()`

## âœ… `.read()`

Reads entire file as string.


You can limit size:

```python

```

---

## âœ… `.readline()`

Reads one line at a time.

```python

```

---

## âœ… `.readlines()`

Returns list of lines.

```python

```

Example result:


---

## âœ… Best Way (Loop Through File)

```python

```

Memory efficient for large files.

# ğŸ”¹ 8ï¸âƒ£ Exception Handling (Basic)

Files may not exist.

---

## âŒ Problem

```python
open("missing.txt")
```

Raises:

```
FileNotFoundError
```

---

## âœ… Solution

```python
try:
    with open("missing.txt") as f:
        content = f.read()
except FileNotFoundError:
    print("File not found")
```

---

## Multiple Exceptions

```python
try:
    ...
except FileNotFoundError:
    ...
except PermissionError:
    ...
```

---

## Optional `finally`

```python
try:
    f = open("data.txt")
except:
    print("Error")
finally:
    print("Done")
```

`finally` always runs.

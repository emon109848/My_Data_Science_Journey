# ğŸ§  Interview-Level Explanation

Internally:

* Lambda creates a function object
* Just like `def`
* But without binding to a name (unless assigned)

Example:

```python
f = lambda x: x+1
```

Is same as:

```python
def f(x):
    return x+1
```

Except lambda has no docstring and limited body.

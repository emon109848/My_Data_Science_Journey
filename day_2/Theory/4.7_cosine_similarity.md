## 1Ô∏è‚É£ Definition

* Measures **similarity between two vectors** based on the **angle between them**
* Value ranges: `-1` (opposite) ‚Üí `0` (orthogonal) ‚Üí `1` (same direction)

[
\text{cosine_similarity}(a,b) = \frac{a \cdot b}{|a| |b|}
]

* Uses **dot product** and **vector norms**

---

## 2Ô∏è‚É£ NumPy Implementation

```python
import numpy as np

a = np.array([1,2,3])
b = np.array([4,5,6])

cos_sim = np.dot(a,b) / (np.linalg.norm(a) * np.linalg.norm(b))
print(cos_sim)  # 0.9746
```

* **Dot product:** numerator ‚Üí measures alignment
* **Norms:** denominator ‚Üí normalize magnitudes

---

## 3Ô∏è‚É£ Angle Between Vectors

* Can get **angle in radians or degrees**:

```python
angle = np.arccos(cos_sim)          # in radians
angle_deg = np.degrees(angle)       # in degrees
print(angle_deg)
```

* Small angle ‚Üí vectors are similar; large angle ‚Üí dissimilar

## 5Ô∏è‚É£ Quick Tips

* Works for **any dimensional vector**
* Normalize vectors first if needed ‚Üí faster computation
* Cosine similarity ignores **magnitude**, focuses on **direction**

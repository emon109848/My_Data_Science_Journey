## 1Ô∏è‚É£ Uniform Distribution

* **Definition:** Every value in range `[low, high)` is **equally likely**
* **NumPy Syntax:**

```python
import numpy as np

# 5 samples between 0 and 1
samples = np.random.uniform(0, 1, 5)
print(samples)
```

* **Use Cases:**

  * Simulations where all outcomes are equally likely
  * Random initialization in ML (e.g., weights)

---

## 2Ô∏è‚É£ Normal (Gaussian) Distribution

* **Definition:** Bell-shaped, defined by **mean (`Œº`)** and **std (`œÉ`)**
* **NumPy Syntax:**

```python
samples = np.random.normal(loc=0, scale=1, size=5)  # mean=0, std=1
```

* **Use Cases:**

  * Modeling natural phenomena (heights, errors, noise)
  * Initialization of neural network weights
  * Statistical inference / z-scores

---

## 3Ô∏è‚É£ Binomial Distribution

* **Definition:** Number of successes in `n` trials with probability `p`
* **NumPy Syntax:**

```python
samples = np.random.binomial(n=10, p=0.5, size=5)
print(samples)  # # successes in 10 trials
```

* **Use Cases:**

  * Coin toss simulations
  * Predicting yes/no outcomes
  * Risk analysis in finance

---

## 4Ô∏è‚É£ Law of Large Numbers (LLN)

* **Idea:** Average of random samples **approaches expected value** as sample size increases

```python
samples = np.random.normal(0,1,100000)
mean = np.mean(samples)  # ‚âà 0
```

* **Use Cases:**

  * Monte Carlo simulations
  * Estimating probabilities
  * Experimental validation

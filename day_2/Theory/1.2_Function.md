# ğŸ”¹ 9ï¸âƒ£ `return` vs `print`

| `return`         | `print`               |
| ---------------- | --------------------- |
| Sends value back | Displays output       |
| Can store result | Cannot store result   |
| Ends function    | Does not end function |

Example:

```python
def add(a, b):
    print(a+b)
```

You cannot do:

```python
x = add(2,3)   # x will be None
```

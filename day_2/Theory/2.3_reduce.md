# ğŸ§  What is `reduce()`?

* Part of **`functools`** module
* Applies a **binary function** cumulatively to the items of an iterable
* Reduces iterable to a **single value**

# ğŸ”¹ 2ï¸âƒ£ Syntax

* `function` â†’ takes **two arguments**
* `iterable` â†’ list, tuple, etc.

* Returns **single value**

# ğŸ”¹ 7ï¸âƒ£ Lazy vs Immediate

* `reduce()` **computes immediately**
* Unlike `map()` or `filter()`, returns **single final result**

* Functional style (reduce) â†’ concise

# ğŸ§  INTERNAL WORKING

1. Takes first two elements â†’ applies function
2. Takes result + next element â†’ applies function
3. Continues until end of iterable
4. Returns final value

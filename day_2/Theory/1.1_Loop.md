# ğŸ”¹ 5ï¸âƒ£ Loop Over Dictionary

```python
student = {"name": "Emon", "age": 22}
```

### Keys

```python
for key in student:
    print(key)
```

### Values

```python
for value in student.values():
    print(value)
```

### Key + Value

```python
for key, value in student.items():
    print(key, value)
```
